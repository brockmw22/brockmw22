CSS Selectors & Specificity

Priority Higher Specificity --> Lower Specificity
Selectors ID Class Type
Selectors Uses Weight
1

ID

#Example 		give a specific ID (Use sparingly) adds 1-0-0
2

Class options

  1. .Example
  2. [type="radio"]
  3. [lang|="fr"]
  4. .Example

For each class, attribute selector, or pseudo-class in a matching selector

add 0-1-0 to the weight value.
3

elements

<p>

For each type or pseudo-element in a matching selector

add 0-0-1 to the weight value.
4

Universal

* { : ; } 		matches elements of any type These selectors do not impact the specificity weight value.
5

No Value


:example()

function takes a selector list as its argument,

and selects any element that can be selected by one of the selectors in that list.

 These selectors do not impact the specificity weight value.
6

Combinators

  1. +
  2. >
  3. ~
  4. " "
  5. | |

may make a selector more specific in what is selected

they don't add any value to the specificity weight.
7

Matching

  • Example.Example,
  • Example:Example,
  • :root #Example input:Example {}

The weight is determined by comparing the values in the three columns

from left to right.
